## configtion
config file at ```~/.config/sigmabar/config.ini```
*default* mean if config.ini has no that key, program will use a hardcode value inside
all config:

```width```
set bar width
*default:1920*
example:
```width=1920```

```height```
set bar height
*default:30*
example:
```height=20```
